Parent as Teacher

家長不是旁觀者，而是孩子的第一位探索教練

我們推廣給家長的不是「又一堂孩子的課」，而是一套親子共學方法：家長學會引導，孩子學會從興趣線索裡說出自己在意的事。

親子共學
01

家長先學會不急著給答案

孩子說出任何聯想時，家長先接住，不評論、不否定，讓孩子願意繼續說。

02

用問題帶孩子找到興趣線索

家長只需要問 What、Why、How，孩子會慢慢從喜歡、好奇、願意多做一下的事情裡看見方向。

03

把興趣變成一個小行動

不用一次找到人生志向，先找出本週可以做的一件小事，讓孩子用行動確認自己是不是真的有興趣。

Course Design

如何從興趣找到夢想？親子共學・夢想九宮格

課程時長約 60 分鐘。核心不是逼孩子說出夢想，而是陪孩子找出「我好像有興趣」的線索。AI 是選用輔助，主角永遠是孩子自己的想法。

課前共看

先和孩子一起看一段關於夢想的影片

請家長先陪孩子看完影片，再問：「你剛剛聽到哪一句最有感覺？」不用急著得到標準答案，先讓孩子說出自己的感受。

暖身

先聊一聊，5 分鐘

問題：「如果明天不用上學、不用上班，你最想做什麼？」家長和孩子都回答，先讓彼此熱身。

一

擴散聯想，20 分鐘

在九宮格中央寫下暫定夢想，問孩子：「如果往這個方向走，你會想到什麼？」把蹦出來的詞填進周圍八格。

二

聚斂聯想，20 分鐘

圈出最有感覺的 1 到 2 格，再問：這是什麼？為什麼你會在意？可以怎麼試試看？

三

變成行動，10 分鐘

選出最想試試看的那一格，一起寫下三件現在就能做的小事。

Adult Cross-Disciplinary Vision

第二推廣對象：想跨域的大人

AI 時代需要的能力，不只是學會一個工具，而是把原本的專業延伸到新的表達方式與作品。

概念展示
  • 工程師也可以成為作家，把邏輯能力轉成故事、文字與作品。工程 x 寫作
  • 老師也可以用 AI 設計網站，把教學理念變成可被看見的產品。教育 x 設計
  • 未來人才不是只會單一技能，而是能跨域整合、表達與創造。未來能力
AI

這一區目前只作為網路公開概念。
先不放實際課程，也不開功能。它用來讓大人理解：AI 不是取代專業，而是讓專業跨出去。

GitHub Pages Demo

使用 Google 帳號登入，資料寫入 Google Sheet

家長不用另外設定密碼。登入時由 Google 驗證身分，再由 Apps Script 把姓名與 email 寫入你的 Google Sheet。

!

設定提醒：
發布前請填入 Google OAuth Client ID 與 Apps Script Web App URL，並確認 Apps Script 設為「任何人」可存取。

DB

資料表欄位：
姓名、email、Google 帳號識別碼、頭像、狀態、後台備註、後台標籤。身份不在前台詢問，之後可由後台自行補註。

Google Sign-In

用 Google 帳號進入會員課程

按下 Google 登入後，系統會自動建立或更新你的會員資料，不需要另外輸入密碼。

登入成功後會自動開啟會員課程。Google 只會提供基本資料：姓名、email 與帳號識別碼。

AI 未來探索平台｜親子共學 GitHub Pages 靜態示範頁。帳號資料可寫入 Google Sheet；課程操作資料暫存在使用者自己的瀏覽器。
